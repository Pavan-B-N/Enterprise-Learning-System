## Skills Measured

### 1. Deploy and Manage Azure Stack HCI Infrastructure (25–30%)

- Plan cluster deployment (hardware requirements, networking, Active Directory)
- Deploy Azure Stack HCI clusters (Windows Admin Center, PowerShell)
- Manage cluster nodes (add/remove nodes, updates, firmware)
- Register and manage with Azure (Azure Arc, billing, monitoring)

### 2. Implement and Manage Compute and Storage (25–30%)

- Manage VMs on Azure Stack HCI (create, configure, live migration)
- Implement Storage Spaces Direct (S2D) (pool configuration, tiers, volumes)
- Manage storage (deduplication, ReFS integrity, capacity planning)

### 3. Implement and Manage Networking (20–25%)

- Implement network infrastructure (SDN, Network Controller, Software Load Balancer)
- Configure Network ATC (intent-based networking)
- Implement microsegmentation and network security

### 4. Maintain and Monitor (20–25%)

- Monitor cluster health (Azure Monitor, Windows Admin Center)
- Implement backup and disaster recovery
- Manage updates and patches (Cluster-Aware Updating)
- Troubleshoot cluster issues

## Study Strategy

1. **Hybrid mindset**: Understand Azure integration with on-premises HCI
2. **Storage Spaces Direct**: Key differentiator — understand deeply
3. **SDN**: Software-defined networking concepts and components
4. **Windows Admin Center**: Know the management workflows

## Exam Tips

- Storage Spaces Direct: mirror vs parity, nested resiliency, volumes
- Network ATC simplifies network configuration — know intents
- Azure Arc integration: resource bridge, VM provisioning from Azure
- Cluster-Aware Updating: maintenance windows, update runs
