## Skills Measured

### 1. Describe Artificial Intelligence Workloads and Considerations (15–20%)

- Identify features of common AI workloads (prediction, anomaly detection, NLP, CV)
- Identify responsible AI principles (fairness, reliability, privacy, inclusiveness, transparency, accountability)

### 2. Describe Fundamental Principles of Machine Learning (20–25%)

- Identify common ML types (supervised, unsupervised, reinforcement learning)
- Describe core ML concepts (features, labels, training, validation, evaluation metrics)
- Describe Azure Machine Learning capabilities (AutoML, Designer, Notebooks)

### 3. Describe Features of Computer Vision Workloads (15–20%)

- Identify types of CV solutions (classification, detection, segmentation, OCR)
- Describe Azure AI Vision service capabilities
- Describe Azure AI Face service capabilities

### 4. Describe Features of Natural Language Processing (15–20%)

- Describe NLP workloads (text analysis, sentiment, translation, language understanding)
- Describe Azure AI Language service capabilities
- Describe Azure AI Speech service capabilities

### 5. Describe Features of Generative AI Workloads (15–20%)

- Describe generative AI concepts (LLMs, prompts, grounding, RAG)
- Describe Azure OpenAI Service capabilities
- Describe responsible generative AI (content filters, abuse monitoring)

## Study Strategy

1. **Conceptual focus**: This is fundamentals — understand concepts, not code
2. **AI use cases**: Match the right AI approach to business problems
3. **Responsible AI**: Know all six Microsoft principles
4. **Azure services**: Which service solves which AI problem

## Exam Tips

- Know the difference between classification, regression, and clustering
- Supervised = labeled data, Unsupervised = unlabeled data
- Azure OpenAI: know deployment models and responsible AI features
- Responsible AI is tested throughout, not just one section
- No coding or hands-on experience required for this exam
