# SC-200: Microsoft Security Operations Analyst — Study Guide

## Exam Overview

| Detail | Info |
|--------|------|
| **Certification** | Microsoft Certified: Security Operations Analyst Associate |
| **Exam Code** | SC-200 |
| **Duration** | 100 minutes |
| **Questions** | 40–60 |
| **Passing Score** | 700/1000 |
| **Cost** | $165 USD |
| **Renewal** | Annual |

## Skills Measured

### 1. Mitigate Threats Using Microsoft Defender XDR (25–30%)

- Mitigate threats using Microsoft Defender for Endpoint, Identity, Office 365, Cloud Apps
- Manage cross-domain investigations (incidents, advanced hunting)

### 2. Mitigate Threats Using Microsoft Sentinel (50–55%)

- Design and configure a Microsoft Sentinel workspace
- Manage content (content hub, repositories, analytics rules)
- Configure data connectors and data collection
- Manage analytics rules (scheduled, NRT, fusion, anomaly)
- Perform threat hunting using KQL
- Configure SOAR (automation rules, playbooks)

### 3. Mitigate Threats Using Microsoft Defender for Cloud (15–20%)

- Configure environment settings and plan workload protections
- Remediate security alerts and incidents
- Configure and manage cloud security posture management (CSPM)

## Study Strategy

1. **KQL mastery**: This is critical — practice writing queries from scratch
2. **Sentinel architecture**: Workspace design, data connectors, costs
3. **Incident response**: Understand the full incident lifecycle
4. **Automation**: Logic Apps as playbooks, automation rules

## Exam Tips

- KQL is throughout the exam — know `where`, `summarize`, `join`, `extend`, `project`
- Sentinel analytics rules: know the difference between scheduled, NRT, and fusion
- Data connector types: Syslog, CEF, Azure service connectors, custom
- Defender for Cloud: Secure Score, recommendations, regulatory compliance
