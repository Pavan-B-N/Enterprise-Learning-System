## Skills Measured

### 1. Plan and Manage an Azure AI Solution (15–20%)

- Select appropriate Azure AI services for solutions
- Plan and configure responsible AI principles
- Create and manage AI resources (Cognitive Services, OpenAI)

### 2. Implement Computer Vision Solutions (15–20%)

- Analyze images using Azure AI Vision
- Implement image classification and object detection (Custom Vision)
- Read text from images (OCR) and documents (Document Intelligence)
- Implement facial recognition solutions (Face API)

### 3. Implement Natural Language Processing Solutions (25–30%)

- Analyze text (sentiment, key phrases, entities, PII detection)
- Build conversational AI (Bot Framework, QnA Maker, Language Understanding)
- Translate text and speech

### 4. Implement Knowledge Mining and Document Intelligence (15–20%)

- Implement Azure AI Search (indexes, skillsets, custom skills)
- Implement document processing (Document Intelligence models)

### 5. Implement Generative AI Solutions (15–20%)

- Develop solutions using Azure OpenAI Service
- Implement Retrieval Augmented Generation (RAG)
- Manage prompts and fine-tuning

## Study Strategy

1. **SDK focus**: Know the Python/C# SDK patterns for each AI service
2. **REST APIs**: Understand request/response formats
3. **RAG pattern**: Combine Azure AI Search with Azure OpenAI
4. **Responsible AI**: Know the six Microsoft principles

## Exam Tips

- Know which AI service solves which problem
- Custom Vision vs. Azure AI Vision — when to use pre-built vs custom models
- Azure OpenAI: deployment models, token limits, content filtering
- AI Search: skillsets, knowledge store, semantic ranking
- Document Intelligence: pre-built models vs. custom models
