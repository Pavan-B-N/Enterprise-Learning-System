# SC-900: Microsoft Security, Compliance, and Identity Fundamentals — Study Guide

| Detail | Info |
|--------|------|
| **Certification** | Microsoft Certified: Security, Compliance, and Identity Fundamentals |
| **Exam Code** | SC-900 |

| **Questions** | 40–60 |
| **Passing Score** | 700/1000 |
| **Cost** | $99 USD |
| **Renewal** | Does not expire |

## Skills Measured

### 1. Describe Security and Compliance Concepts (10–15%)

- Describe the shared responsibility model
- Describe defense-in-depth
- Describe Zero Trust model (verify explicitly, least privilege, assume breach)
- Describe encryption and hashing concepts

### 2. Define the Concepts of Identity (25–30%)

- Describe identity concepts (authentication vs authorization, identity providers)
- Describe Microsoft Entra ID (tenant, users, groups, roles)
- Describe authentication methods (MFA, passwordless, Conditional Access)
- Describe access management (RBAC, ABAC)
- Describe identity governance (PIM, access reviews, entitlement management)

### 3. Describe Microsoft Security Solutions (35–40%)

- Describe core Azure security services (DDoS, Firewall, NSG, Bastion)
- Describe security management capabilities (Defender for Cloud, Secure Score)
- Describe security capabilities of Microsoft Sentinel
- Describe threat protection with Microsoft Defender XDR

### 4. Describe Microsoft Compliance Solutions (20–25%)

- Describe Microsoft Purview compliance solutions
- Describe information protection, data lifecycle, insider risk
- Describe eDiscovery and audit capabilities
- Describe Azure governance (Policy, Blueprints, resource locks)

## Study Strategy

1. **Breadth over depth**: Understand what each service does at a high level
2. **Zero Trust**: This concept threads through the entire exam
3. **Service comparisons**: Know which tool solves which problem
4. **Compliance**: Understand the Microsoft Purview suite components

## Exam Tips

- Zero Trust principles appear in many questions as context
- Know Microsoft Defender XDR components (Endpoint, Identity, Office 365, Cloud Apps)
- Understand the difference between Microsoft Sentinel and Defender for Cloud
- Purview: Information Protection, DLP, Insider Risk, Communication Compliance
- This is conceptual — no hands-on experience required
