# AZ-900: Microsoft Azure Fundamentals — Study Guide

## Exam Overview

| Detail | Info |
|--------|------|
| **Certification** | Microsoft Certified: Azure Fundamentals |
| **Exam Code** | AZ-900 |
| **Duration** | 45 minutes |
| **Questions** | 40–60 |
| **Passing Score** | 700/1000 |
| **Cost** | $99 USD |
| **Renewal** | Does not expire |

## Target Audience

This certification is ideal for candidates who are beginning to work with cloud-based solutions and services. It validates foundational knowledge of cloud concepts, Azure services, Azure workloads, security and privacy in Azure, and Azure pricing and support.

## Skills Measured

### 1. Describe Cloud Concepts (25–30%)

- **Cloud computing benefits**: High availability, scalability, elasticity, agility, disaster recovery
- **Cloud service models**: IaaS, PaaS, SaaS — know the shared responsibility for each
- **Cloud deployment models**: Public, Private, Hybrid — when to use each
- **Consumption-based model**: CapEx vs OpEx, pay-per-use pricing

### 2. Describe Azure Architecture and Services (35–40%)

- **Core architectural components**: Regions, Region Pairs, Availability Zones, Resource Groups, Subscriptions, Management Groups
- **Compute services**: VMs, App Service, Container Instances, Azure Kubernetes Service, Azure Functions
- **Networking**: Virtual Networks, VPN Gateway, ExpressRoute, Azure DNS
- **Storage services**: Blob, File, Queue, Table, Disk — access tiers (Hot/Cool/Archive)

### 3. Describe Azure Management and Governance (30–35%)

- **Cost management**: Cost factors, Pricing Calculator, TCO Calculator, Cost Management tool
- **Governance tools**: Azure Policy, Resource Locks, Azure Blueprints, Cloud Adoption Framework
- **Resource management**: Azure Portal, CLI, PowerShell, ARM templates, Azure Arc
- **Monitoring**: Azure Advisor, Service Health, Azure Monitor

## Study Strategy

1. **Week 1**: Cloud Concepts — understand the why before the what
2. **Week 2**: Azure Architecture — hands-on in the Azure Portal (free tier)
3. **Week 3**: Management & Governance — focus on cost and policy
4. **Week 4**: Practice exams and review weak areas

## Exam Tips

- Questions are scenario-based — understand "when to use what"
- No deep technical configuration required
- Focus on understanding trade-offs (cost vs. performance vs. availability)
- Know the difference between Azure services at a high level
- Azure AD is now Microsoft Entra ID — exam may use either name
