# MS-700: Managing Microsoft Teams — Study Guide

## Exam Overview

| Detail | Info |
|--------|------|
| **Certification** | Microsoft Certified: Teams Administrator Associate |
| **Exam Code** | MS-700 |
| **Duration** | 100 minutes |
| **Questions** | 40–60 |
| **Passing Score** | 700/1000 |
| **Cost** | $165 USD |
| **Renewal** | Annual |

## Skills Measured

### 1. Plan and Configure a Microsoft Teams Environment (35–40%)

- Plan and configure network settings for Microsoft Teams
- Plan and configure Teams governance and lifecycle (naming policy, expiration, templates)
- Plan and configure external and guest access
- Plan and configure security and compliance (DLP, retention, eDiscovery, barriers)

### 2. Manage Chat, Channels, and Apps (25–30%)

- Manage messaging policies and compliance
- Manage channels (standard, private, shared)
- Manage apps (permission policies, setup policies, custom apps)
- Manage collaboration experiences (tabs, connectors, bots)

### 3. Manage Meetings, Calling, and Devices (30–35%)

- Manage meeting experiences (policies, configurations, live events, webinars)
- Manage Phone System (calling plans, direct routing, operator connect)
- Manage Teams devices (provisioning, configuration, monitoring)
- Monitor and report on Teams (Call Quality Dashboard, usage reports)

## Study Strategy

1. **Admin Center**: Navigate and configure policies hands-on
2. **Policies**: Understand the hierarchy (global → group → user)
3. **Voice**: Know the three PSTN connectivity options
4. **Network planning**: Bandwidth requirements and QoS

## Exam Tips

- Policy assignment order: user-level overrides group-level overrides global
- Direct Routing requires SBC (Session Border Controller)
- Shared channels use Azure B2B direct connect (not guest access)
- Information barriers prevent specific groups from communicating
- Teams Premium features: custom branded meetings, meeting templates, watermarks
