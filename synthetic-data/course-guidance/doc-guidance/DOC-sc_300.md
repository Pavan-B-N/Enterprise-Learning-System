# SC-300: Microsoft Identity and Access Administrator — Study Guide

| **Cost** | $165 USD |
| **Renewal** | Annual |

## Skills Measured

### 1. Implement Identities in Microsoft Entra ID (20–25%)

- Configure and manage a Microsoft Entra tenant
- Create, configure, and manage identities (users, groups, devices)
- Implement and manage external identities (B2B, B2C)
- Implement and manage hybrid identity (Azure AD Connect, cloud sync)

### 2. Implement Authentication and Access Management (25–30%)

- Plan, implement, and manage MFA and passwordless authentication
- Plan and implement Conditional Access policies
- Manage Microsoft Entra Identity Protection (risk policies, sign-in risk)

### 3. Implement Access Management for Applications (15–20%)

- Manage and monitor application access (enterprise apps, app registrations)
- Plan and implement app registrations (permissions, API exposure, certificates)

### 4. Plan and Implement Identity Governance (25–30%)

- Plan and implement Privileged Identity Management (eligible vs active roles)
- Plan and implement entitlement management (access packages, catalogs)
- Plan and implement access reviews (recurring, self-review, guest access)

## Study Strategy

1. **Conditional Access**: This is the core of the exam — master policy configuration
2. **PIM**: Know the activation workflow, approval, and JIT access
3. **Hybrid**: Understand sync topologies and authentication methods
4. **Governance**: Access reviews, entitlement management, lifecycle workflows

## Exam Tips

- Conditional Access: conditions × grant controls × session controls — know all options
- Authentication methods: FIDO2, Microsoft Authenticator, Windows Hello, phone sign-in
- PIM: eligible vs active assignments, activation requirements
- Hybrid identity: password hash sync vs pass-through auth vs federation
