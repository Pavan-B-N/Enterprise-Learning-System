> **Purpose:** Assessment Agent scoring reference

## Scoring Framework

### Score Interpretation

| Score Range | Readiness Level | Recommendation |
|-------------|----------------|----------------|
| 85-100% | Ready | Schedule exam within 1 week |
| 75-84% | Strong | Schedule exam within 2 weeks, light review |
| 70-74% | Borderline | 1-2 more weeks of targeted study on weak areas |
| 60-69% | Almost Ready | 3-4 more weeks, revisit fundamentals in weak domains |
| 45-59% | Developing | Extend plan by 4-6 weeks, consider prerequisite gaps |
| 0-44% | Not Ready | Major gaps — reassess plan, consider foundational cert first |

### Question Difficulty Weighting

| Difficulty | Points | Typical Allocation |
|-----------|--------|-------------------|
| Basic (recall) | 1 point | 20% of questions |
| Intermediate (application) | 2 points | 50% of questions |
| Advanced (analysis/design) | 3 points | 30% of questions |

### Domain Coverage Rules

Each practice assessment MUST cover all skill domains for the target certification proportionally to exam weights. The Assessment Agent should:

1. Select questions from each domain according to CERT_SKILL weights
2. Include at least 1 question per domain (minimum coverage)
3. Weight harder questions toward declared weak areas (adaptive difficulty)
4. Ensure each question has a grounding citation from approved documents

### Passing Criteria

1. **Overall score ≥ 70%** (weighted by difficulty)
2. **No domain below 50%** (must demonstrate baseline in all areas)
3. **Minimum 2 advanced questions correct** (proves depth, not just breadth)

### Readiness Assessment Triggers

The system should recommend exam scheduling when:
- 3 consecutive practice assessments score ≥ 75%
- All domains individually score ≥ 60%
- Learner has completed ≥ 80% of study plan milestones
- Total study hours meet minimum for certification level

### Remediation Loop Triggers

The system should recommend returning to study when:
- Score drops below 65% after previously scoring ≥ 70% (regression)
- Any single domain scores < 40% (critical gap)
- Learner fails to improve across 2 consecutive attempts
- Time gap between assessments > 3 weeks (knowledge decay risk)

## Citation Requirements for Generated Questions

Every question generated by the Assessment Agent MUST:
1. Reference at least one approved knowledge document
2. Include the document ID and section in the citation
3. Provide the correct answer with an explanation grounded in the cited material
4. Not hallucinate content — all technical claims must be traceable to source docs

### Citation Format
```
Source: [DOC-ID] - [Section Name]
Example: DOC-AZ204-GUIDE - Pattern 2: Managed Identity
```

## Fairness and Bias Controls

1. Questions should not assume specific cultural context
2. Scenario descriptions should use diverse but clearly synthetic identifiers
3. Difficulty should not correlate with specific demographic patterns
4. All learners at the same readiness level should receive equivalent-difficulty assessments
5. Question bank should be periodically reviewed for outdated or misleading content
